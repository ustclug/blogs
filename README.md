# blogs

USTCLUG 同学们的博客列表，参考了 TUNA 协会的 [blogroll](https://github.com/tuna/blogroll) 项目。

可以通过发 Issues 或 Pull requests 的方式加入此列表，需要附上你的博客的名字、URL 与 RSS 订阅方式，如果能介绍一下你的博客的内容就更好了。
